## 安装
> 启用该扩展前, 项目必须安装[spatie/laravel-backup](https://github.com/spatie/laravel-backup)并完成配置

1. `composer require zwping/backup`
2. 扩展中更新&启用
3. 配置用户菜单&权限
4. 打开 http://your-host/admin/backup
